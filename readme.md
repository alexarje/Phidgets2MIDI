# Phidgets2MIDI

Version: 	   0.1  
Created date:  2011-01-12  
Modified date: 2011-01-12


## What can it do?

Phidgets2MIDI is an application for creating MIDI messages from a Phidgets sensor interface.


## Usage 

Connect Phidgets kit, open patch, play around.
